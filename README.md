# Compile

 - Download Rust Nightly with rustup.
 - Install allegro with your package manager or build it.
 - Build with cargo.
 - Run with our [Launcher](https://github.com/Litecrafty/Launcher) or manually with
```bash
cargo run -- test_token
```
